# Core Scripting Constructs and Control Flow

This repository contains Bash shell scripts that demonstrate core scripting concepts and control flow in Linux.

## Scripts Included

### calculator.sh
Performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

### filecheck.sh
Checks whether a file exists or not.

### numbercheck.sh
Checks whether a number is:
- Positive
- Negative
- Zero

- Linux
